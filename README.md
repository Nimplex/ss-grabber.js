# ss-grabber.js
Discord.js bot that sends images from https://prnt.sc to discord chat. 

## Requirments: 
```
discord.js ^12.3.1 
node-fetch ^2.6.1
```

## Guide:

Install all dependencies
`npm install`

Your bot token should be in config.js file - it should look something like this
```js
module.exports = {
  token: 'token-here',
  prefix: '.'
}
```

To run bot use
`node src/index.js`
